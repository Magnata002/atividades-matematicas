# Kit 600 Atividades Matemáticas - Página de Vendas

Este projeto é uma página de vendas moderna, futurista e responsiva para um "Kit de Atividades Matemáticas".

## Estrutura do Projeto

- **index.html**: A estrutura principal da página com todo o copy de vendas.
- **style.css**: Estilos com tema escuro, efeitos de neon, glassmorphism e animações.
- **script.js**: Lógica para animações de scroll e interatividade.

## Como Visualizar

1. Abra a pasta do projeto.
2. Dê um duplo clique no arquivo `index.html` para abrir em seu navegador padrão.

## Tecnologias Utilizadas

- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid, Animações)
- JavaScript (Intersection Observer API)
- FontAwesome (Ícones)
- Google Fonts (Outfit)

## Personalização

- Para alterar as cores, edite as variáveis `:root` no início do arquivo `style.css`.
- Para editar o texto, modifique o arquivo `index.html`.
